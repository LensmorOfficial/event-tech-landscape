# Contributing

Thank you for contributing to the Event Tech Landscape.

## How to contribute

- Open an issue using the tool submission template.
- Submit a pull request with additions or corrections.

## Tool inclusion criteria

- The tool is active and publicly available.
- The website link is official and working.
- The category is the best fit for the tool's primary use case.
- Avoid duplicate entries across categories unless the tool has distinct capabilities.

## Formatting rules

- Keep entries consistent with the table format.
- Use a concise description of category focus.
- Pricing model should be one of: Freemium, Usage-based, Subscription, Enterprise.

## Review process

All submissions are reviewed for accuracy, relevance, and clarity.
